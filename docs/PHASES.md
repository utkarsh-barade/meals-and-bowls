# Development Phases

# 🍽️ Meals & Bowls - Project Roadmap

Version: 1.0

---

# Goal

Break the project into small, manageable phases so each phase produces a working application.

---

# Phase 1 — Project Setup

## Objectives
- Initialize project
- Configure authentication
- Create base layout
- Set up navigation

## Deliverables
- Admin Login
- Customer Login
- Customer Sign Up
- Dashboard layout
- Shared UI components

**Status:** Foundation Complete

---

# Phase 2 — Customer Management

## Objectives
Build customer management.

## Features
- Customer List
- Add Customer
- Edit Customer
- View Customer Details
- Customer Photo Upload
- Search Customer

## Customer Data
- Full Name
- Mobile Number
- Photo

**Milestone**
Admin can manage customers.

---

# Phase 3 — Plan & Subscription Management

## Objectives
Manage subscriptions.

## Features
- Silver Plan
- Gold Plan
- Assign Plan
- Renew Plan
- Active Subscription
- Expiry Calculation

**Business Rules**
- One active subscription per customer.
- Plan expires by meals consumed or validity.

**Milestone**
Customer subscriptions are fully managed.

---

# Phase 4 — Meal Management

## Objectives
Build the core business workflow.

## Features
- Search Customer
- Serve Lunch
- Serve Dinner
- Prevent duplicate meal entries
- Automatically update meals consumed
- Calculate remaining meals

**Milestone**
Daily meal serving is operational.

---

# Phase 5 — Meal History

## Objectives
Support correction of missed meal entries.

## Features
- View Meal History
- Correct Meal
- Prevent duplicate corrections
- Update meal counts
- Record correction history

**Milestone**
Meal records can be corrected safely.

---

# Phase 6 — Payments

## Features

---

# MVP Completion Criteria

The MVP is complete when:

- Admin can manage customers.
- Admin can assign and renew plans.
- Admin can serve Lunch and Dinner.
- Meal deductions are accurate.
- Admin can correct missed meal entries.
- Payments are recorded.
- Customers can view their dashboard.
- WhatsApp notifications are working.
- Reports are available.
